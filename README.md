# 🏜️ واحة هايرول (Hyrule Oasis)

> **رحلتك إلى هايرول تبدأ من هنا. أعظم ملحمة في تاريخ الألعاب.. بأمان تام وبدون أي حرق.**

مرحباً بك في **واحة هايرول**، الملاذ الأول والوحيد عربياً المخصص لمحبي وعشاق سلسلة ألعاب **أسطورة زيلدا (The Legend of Zelda)**، والموجه للباحثين عن نقطة دخول آمنة وخالية من الحرق (Zero Spoilers).

بنيت هذا الموقع ليكون رسالة حب وشغف لأسطورة زيلدا، ليساعد أي شخص ودّه يدخل هذا العالم الساحر ويضيع بين ألعابه، خطوطه الزمنية، وأساطيره، دون الخوف من تخريب القصة أو المفاجآت.

---

## ✨ مميزات المشروع والصفحات المستكشفة

- 🎮 **[مكتبة الألعاب المرجعية (`/games`)]:** أرشيف متكامل لجميع ألعاب زيلدا الأساسية مرتبة وخالية من الحرق، مع قدرة البحث والتصفية حسب العصر والأجهزة.
- 📜 **[موسوعة هايرول (`/universe`)]:** أساسيات الأسطورة (الـ Lore) والأعراق، الأسلحة، والقوى الكونية. مصممة كبطاقات زجاجية تفاعلية تعطي شعور القراءة من كتاب أثري سحري.
- 🧭 **[أدلة تشغيل المحاكيات (`/guide/emulation`)]:** مقالات وشروحات حصرية تشرح خطوة بخطوة كيفية تشغيل هذه الألعاب على أجهزة الـ **PC** أو **الجوال (iOS / Android)**.
- 🎯 **[الاختبار الذكي (`/quiz`)]:** نظام ترشيح ديناميكي يسألك عن توفر أجهزتك واهتماماتك الجوية ليختار لك "لعبتك القادمة" الأنسب لشخصيتك!
- ⏳ **[الخريطة الزمنية (`/timeline`)]:** استعراض بصري وتسلسل فريد لأحداث السلسلة وتفرعاتها عبر الزمن (في طور الإضافة والتطوير المستمر).

---

## 🛠️ كيف تم بناء الواحة؟ (التقنيات المستخدمة)

استخدمت في هذا المشروع أحدث التقنيات لضمان أفضل أداء للزوار وأروع شكل بصري ممكن (Glassmorphism & Magic Aesthetics):

- **[Next.js 16 (React)](https://nextjs.org/)** - إطار العمل القوي لصفحات سريعة و Server-Side Rendering.
- **[Framer Motion](https://www.framer.com/motion/)** - حجر الزاوية لكل الأنميشنز والحركات السلسة عبر الموقع (Scroll Animations & Hero Orbits).
- **TypeScript** - لكتابة كود قوي وموثوق وتلافي الأخطاء.
- **CSS Modules & Custom CSS** - ستايلات مخصصة لكل صفحة وبناء واجهة بتأثيرات الـ Glassmorphism المعاصرة.

## 🚀 طريقة التشغيل وتطوير المشروع محلياً

إذا حبيت تستكشف الكود بنفسك، أو تشغله ليتعلم منه مجتمع المطورين:

```bash
# 1. قم بتحميل المستودع
git clone https://github.com/UIUO0/HyruleOasis.git
cd HyruleOasis

# 2. قم بتثبيت الحزم المطلوبة
npm install

# 3. شغل السيرفر المحلي
npm run dev
```

افتح المتصفح على [http://localhost:3000](http://localhost:3000) واستمتع بالمغامرة.

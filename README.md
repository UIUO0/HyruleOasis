> *"رحلتك إلى هايرول تبدأ من هنا. أعظم ملحمة في تاريخ الألعاب.. بأمان تام وبدون أي حرق."*

مرحباً بك في **واحة هايرول**، المشروع الذي صُنع بشغف كرسالة حب لعالم **أسطورة زيلدا (The Legend of Zelda)**. هذا الموقع هو نقطة الدخول والموسوعة العربية الأمثل لكل شخص يود التعمق في هذا العالم الساحر، خطوطه الزمنية، ألعابه، وأساطيره، مع ضمان تجربة خالية تماماً من تخريب القصة (Zero Spoilers).

---

## ✨ المميزات والخدمات (Features)

يقدم المشروع تجربة استكشاف متكاملة عبر عدة أقسام رئيسية صُممت بعناية:

### 🎮 مكتبة الألعاب المرجعية الأضخم (`/games`)
أرشيف متكامل لجميع أجزاء سلسلة زيلدا الأساسية. تعرّف على كل لعبة من خلال بطاقات أنيقة توفر:
- نبذات قصصية مكتوبة بعناية (بدون حرق).
- روابط تحميل مباشرة آمنة لتجربة الألعاب.
- فلاتر متقدمة للبحث حسب سنة الإصدار أو الجهاز (Platform).

### 📜 الموسوعة الشاملة (Lore Universe) (`/universe`)
تغوص هذه الصفحة في الأساسيات وأعماق أسطورة هايرول. اقرأ واستكشف تاريخ:
- الأعراق (The Races) المتواجدة في هايرول (زورا، غورون، ريتو، جيرودو..إلخ).
- الأسلحة الأسطورية وسيف الدمار.
- *تصميم بصري فريد باستخدام Glassmorphism لتشعر وكأنك تقرأ من كتاب سحري قديم.*

### ⏳ الخريطة الزمنية التفاعلية (`/timeline`)
استعراض بصري وتسلسل زمني مبسط لأحداث السلسلة وتفرعاتها (Timelines)، يساعد الوافدين الجدد على فهم تسلسل الألعاب وكيف ترتبط ببعضها قِبَل وبعد انقسام الخط الزمني. *(مزود بخلفية تفاعلية للجسيمات السحرية - Particles Background).*

### 🧭 أدلة شاملة للمحاكيات (`/guide/emulation`)
شروحات تقنية مبسطة وحصرية خطوة بخطوة للعب السلسلة على مختلف الأجهزة:
- **PC:** المحاكيات الأقوى مثل (Ryujinx, Cemu, RetroArch, Lime3DS).
- **Mobile (iOS & Android):** محاكيات الهواتف الذكية مثل (Sudachi, Suyu, Delta, Folium).

### 🎯 نظام "اختبار لعبتك القادمة" الذكي (`/quiz`)
ترددت وما عرفت من وين تبدأ؟ صممنا محرك ترشيح ديناميكي يسألك عن (الأجهزة المتوفرة لديك، تفضيلاتك في اللعب، نمط الرسم الذي تفضله) ليقوم بتحليل إجاباتك ويختار لك "اللعبة الأمثل" لتبدأ بها رحلتك.

## 🎨 واجهة المستخدم وتجربة الاستخدام (UI/UX)

تم بناء **واحة هايرول** للتركيز على الجماليات والتفاعلية:
- **نظام Zero Spoilers الآمن:** استخدام مكون `<SpoilerText />` مخصص يقوم بإخفاء وتغبيش (Blur) أي نصوص قد تحتوي على حرق بشكل تدريجي باستخدام تأثير `GradualBlur`.
- **الـ Glassmorphism:** اعتماد مكثف على التأثيرات الزجاجية، الظلال الملونة (Glow)، والشفافية لإعطاء مظهر متطور وعصري.
- **الحركة التفاعلية:** الاستفادة القصوى من `Framer Motion` لتقديم حركات سلسة عبر الصفحات، تأثيرات التمرير (Scroll Animations)، وظهور المكونات (Fade-ins).

## 🛠️ التقنيات المستخدمة (Tech Stack)

تم بناء المشروع باستخدام أحدث تقنيات الويب لضمان أداء عالٍ (High Performance) وتجربة مستخدم لا تشوبها شائبة:

- **[Next.js 14+ (App Router)](https://nextjs.org/):** المحرك الأساسي للمشروع، تم استخدامه لتقديم صفحات سريعة مع Server-Side Rendering (SSR) وضمان استهلاك ممتاز للـ SEO.
- **[React 18](https://react.dev/):** لبناء واجهات المستخدم والمكونات التفاعلية القابلة لإعادة الاستخدام.
- **[TypeScript](https://www.typescriptlang.org/):** ضمان جودة الكود، أمان الأنواع (Type-Safety)، وتجنب الأخطاء البرمجية الشائعة.
- **[Framer Motion](https://www.framer.com/motion/):** المكتبة العملاقة خلف كل الحركات والانميشنز السلسة في الموقع.
- **Tailwind CSS & Custom CSS Modules:** لدمج سرعة التطوير مع التخصيص العميق للتصاميم المعقدة (أوراق الأنماط المخصصة).
- **Vercel:** استضافة سحابية فائقة السرعة.

## 🚀 التشغيل والتطوير محلياً (Getting Started)

إذا كنت مطوراً وترغب في استكشاف الكود البرمجي الخاص بالواحة وتجربته على جهازك الشخصي:

```bash
# 1. قم باستنساخ المستودع (Clone the repository)
git clone https://github.com/UIUO0/HyruleOasis.git

# 2. ادخل إلى مجلد المشروع
cd HyruleOasis

# 3. قم بتثبيت الحزم والاعتمادات المطلوبة (Install packages)
npm install
# أو باستخدام yarn
yarn install

# 4. ابدأ بتشغيل سيرفر التطوير المحلي
npm run dev
# أو باستخدام yarn
yarn dev
```

الآن افتح متصفحك على [http://localhost:3000](http://localhost:3000) واستمتع برحلتك داخل الكود البرمجي للموقع.

# صور ألعاب Zelda

هذا المجلد مخصص لصور الألعاب (Covers / Key Art / Screenshots) لاستخدامها عبر الموقع لاحقاً.

## المجلدات المقترحة لكل لعبة

- `public/images/games/<slug>/cover.webp` (الصورة الرئيسية)
- `public/images/games/<slug>/hero.webp` (للهيرو أو الخلفيات)
- `public/images/games/<slug>/shots/01.webp` (لقطات)

## تسمية الـ slug

- أحرف صغيرة + شرطات فقط (kebab-case)
- بدون مسافات أو رموز خاصة أو عربي
- أمثلة:
  - `breath-of-the-wild`
  - `tears-of-the-kingdom`
  - `twilight-princess`
  - `wind-waker`
  - `ocarina-of-time`

## الامتدادات المفضلة

- الأفضل: `webp`
- المقبول: `jpg` / `png`

## توصية حجم وجودة

- غلاف: عرض 900–1400px
- Hero: عرض 1600–2400px
- حاول تخلي الملف أقل من 500KB للغلاف وأقل من 900KB للهيرو إذا تقدر

## ملاحظة حقوق الاستخدام

استخدم صور عندك حق استخدامها (تصويرك/مصادر مسموحة/إذن) قبل رفعها للريبو.
